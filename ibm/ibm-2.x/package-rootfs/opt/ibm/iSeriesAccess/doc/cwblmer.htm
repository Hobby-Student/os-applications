CWBLM0001[Contents]

Cause
A release license was attempted on a license that is in use and cannot be released.

Recovery
A license can only be released when it is not in use. Shut down all licensed programs before attempting to release the license.

CWBLM0002[Contents]

Cause
A release license was attempted on a system that does not currently have a license in use.

Recovery
None

CWBLM0003[Contents]

Cause
The license usage limit has been exceeded but the license has been granted. The maximum number of available licenses for the product are already in use. However, you may continue to use the product.

Recovery
Continue processing. You are, however, out of compliance with the licensing agreement. Contact your server administrator to have the license agreement adjusted.

CWBLM0004[Contents]

Cause
The license limit has been exceeded but the request is within the grace period. The maximum number of available licenses for the product are already in use. However, you may continue to use the product during the grace period.

Recovery
Continue processing. Contact your server administrator and have the license agreement adjusted before the grace period expires. Otherwise, you may be prevented from using the product in the future.

CWBLM0011[Contents]

Cause
Internal application error.

Recovery
The application software is in error. Make sure you have applied the latest code levels.

CWBLM0012[Contents]

Cause
The license request failed because the server information was not in the license memory at the time of the request.

Recovery
The application software is in error. Make sure you have applied the latest code levels.

CWBLM0013[Contents]

Cause
A communications error occurred. No further information is available.

Recovery
It is possible that network communication has failed on your PC. Make sure all network cables are connected. If the problem persists, contact you network administrator for assistance.

CWBLM0014[Contents]

Cause
Licenses can only be granted when connecting to V3R2 or later versions of iSeries.

Recovery
Try again with a later release of iSeries.

CWBLM0015[Contents]

Cause
The license request failed because an incorrect license handle was used.

Recovery
The application software is in error. Make sure you have applied the latest code levels.

CWBLM0017[Contents]

Cause
Unable to access server licensing information. The following contains more information, depending on the rc2 number displayed in the error message you received:
rc2 = 28 License information was not found.
rc2 = 29 License information was not available.

Recovery
Make sure that iSeries Access has been installed on the server.

CWBLM0018[Contents]

Cause
The server could not process the license request. The following contains more information, depending on the rc2 number displayed in the error message you received:
rc2 = 01; An invalid request was sent to the host.
rc2 = 31; More than one release was found.
rc2 = 33; Incorrect release format specified.

Recovery
The application software is in error. Make sure you have applied the latest code levels.

CWBLM0019[Contents]

Cause
The host is returning an error to the license client. The following contains more information, depending on the rc2 number displayed in the error message you received:
rc2 = 25; Request license error.
rc2 = 26; Release license error.
rc2 = 27; Retrieve license information error.
rc2 = 2A; The release handle does not match the request handle.

Recovery
The application software is in error. Make sure you have applied the latest code levels.

CWBLM0020[Contents]

Cause
The maximum number of available licenses for iSeries Access are currently in use.

Recovery
Wait until a license is released and try again.

rc2 = 21; Invalid product ID.
rc2 = 22; Invalid release.
rc2 = 23; Invalid feature.
rc2 = 24; Invalid type of license information.

Recovery
The application software is in error. Make sure you have applied the latest code levels.
